   32     /******************************************************************************
   33      *   Function:  gpi_pin_read_564xl_fnc
   34        @brief    Read data from GPI Pin Data Input Registers .
   35        @details  Read data from GPI Pin Data Input Registers
   36   
   37        @param[in]     pcr_num   - GPIO Pin Data Input Register number
   38   
   39        @return The read data

   42     {

   44   
   45       /* Read data from GPI Pin Data Input Registers  */

   52        @brief    GPO Write Data function.

   54

   58        @return none

   63       SIU.GPDO[pcr_num].R = data;
